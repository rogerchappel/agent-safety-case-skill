# Security Policy

## Supported Versions

The current `main` branch is the supported development line for security fixes.

## Reporting a Vulnerability

Please report suspected vulnerabilities through GitHub security advisories or by opening a minimal issue that omits exploit details.

Include the affected version, reproduction steps, expected impact, and any suggested remediation. Do not include private plans, credentials, customer data, or live approval records.

## Scope

`agent-safety-case-skill` is local-first. It reads local plans and writes reviewable safety cases without approving, publishing, sending, or calling external services.
